# Habit Tracker

A simple weekly habit tracker built with Vanilla HTML, CSS, and JavaScript.

## Features

- Add habits
- Edit habits
- Delete habits
- Weekly habit tracking
- Previous and next week navigation
- Current week shortcut
- Current day highlight
- Consecutive streak tracking
- Local storage persistence
- Responsive layout
- Empty state handling

## Tech Stack

- HTML
- CSS
- JavaScript

## How to Run

1. Clone the repository

```bash
git clone <your-repo-url>
```

2. Open the project folder

```bash
cd habit-tracker
```

3. Run the project

Simply open `index.html` in your browser.

No build tools or installation required.

## Features Overview

### Habit Management

Users can:

- Add habits
- Edit habits
- Delete habits

### Weekly Navigation

Users can:

- Move to previous weeks
- Move to next weeks
- Return to current week

### Habit Tracking

- Each habit has daily checkboxes
- Checkboxes persist using localStorage
- Current streak is calculated automatically

### Responsive Design

The layout adjusts for:

- Mobile screens
- Tablet screens
- Desktop screens

## Local Storage

All habits and completed dates are stored in browser localStorage.

Data remains available after page refresh.
